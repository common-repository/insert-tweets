Search for tweets related to your searches and automatically insert them into your pages or posts. Set hours, tweets number, keywords and more.

== Description ==

On each page or post you can write the search text for tweets. Configure every few hours post or a page is automatically updated and if you use the articles titles or search parameter. Dates of post and pages will be updated with the current date of update.
<br><br><strong>Features</strong>
<ul>
<li>Customize insert options</li>
<li>No api key type is needed</li>

</ul>

== Installation ==

This section describes how to install the plugin and get it working.

1. Install the plugin via the plugins menu in your administrator.
2. Activate it and you'll see a new menu option in "Settings" the "Search tweets". 
3. Configure hours and options
